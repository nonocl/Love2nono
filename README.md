# PFP Gallery

Galerie locale pour organiser tes photos de profil Discord.

✨ Fonctionne 100% hors-ligne après le premier chargement
🎨 Deux thèmes : Girly (rose) et Sombre (violet)
🏷️ Catégories illimitées avec couleurs et icônes
💾 Stockage local sur ton appareil (IndexedDB)
📱 Compatible iOS, Android, Desktop

## Utilisation

Va sur l'URL du site, fais "Ajouter à l'écran d'accueil" depuis Safari (iOS) ou Chrome (Android), et utilise comme une vraie app.
